NAME

logstats - log statistics to file  

SYNOPSIS

filterset logstats
{
    show "statistic1"
    show "statistic2"
}
 

DESCRIPTION

The logstats filter-set writes statistics to the BuGLe log (see bugle-log(3)). The statistics are gathered from other filter-sets, which must be listed in the chain ahead of logstats.

The statistics to display are selected by the show pseudo-variable in the filters file. These reference statistics that are defined in ~/.bugle/statistics. Most users should just copy this file from the example in the BuGLe documentation; it lists most of the statistics that one might wish to use.  

BUGS

No statistics are reported for the first frame.  

FILES

$HOME/.bugle/statistics
Formulae and metadata for statistics
 

ENVIRONMENT

BUGLE_STATISTICS
If set, defines the location of the statistics file instead of $HOME/.bugle/statistics.
